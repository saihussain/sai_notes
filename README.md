# 📒 Sai Notes

A premium Xiaomi-inspired Notes & Productivity App built using Flutter.

---

# ✨ Features

## 📝 Notes System

* Create notes
* Edit notes
* Delete notes
* Pin notes
* Favorite notes
* Archive notes
* Search notes instantly
* Compact premium UI

## 📚 Notebook Organization

* Multiple notebooks
* Organize notes by category
* Notebook-based productivity workflow
* Premium notebook cards

## ✅ To-do System

* Create tasks
* Mark tasks completed
* Pending & completed sections
* Xiaomi-style compact task UI

## 🔍 Smart Search

* Real-time note search
* Search titles & content
* Dedicated search screen

## ⚙️ Settings

* Dark mode UI
* AMOLED-inspired design
* Backup options
* Grid view preparation
* Modern settings page

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/saihussain/sai_notes.git
```

## Open Project

```bash
cd sai_notes
```

## Install Dependencies

```bash
flutter pub get
```

## Run App

```bash
flutter run
```
